# geocell

⚠️ **This project is currently under active development.**

A high-performance geospatial library for Python that provides unified access to multiple geocoding systems including:

- Web Mercator tiles (XYZ)
- Quadkeys
- H3 hexagonal hierarchical indexing
- S2 geometry

The library features Cython-optimized implementations with support for vectorized operations and pandas integration.

---

_This is a work in progress. APIs may change without notice._
